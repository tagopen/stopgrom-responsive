vendors - в этой папке должны быть плагины, т.е. любой код, написанный не тобой; Например: bootstrap.sass
themes - здесь хранятся шаблоны сайтов. Например, agency в бутсрапе.
pages - сюда нужно добавлять файлы отдельных страниц. Например, страница "О нас" или "Контакты"
Layout - тут хранятся отдельные блоки: меню, форма - отправить заказ, футер, слайдер и др.
utils - глобальные правила - миксины, функции, анимация, то что позволяет обрабатывать код. Например, тот же круглый progress-bar.
base - там, расположен код, который будет 100% использоваться в каждом проекте. Например, сброс стилей, прибитый футер к низу экрана.
